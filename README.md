## Projetos em Destaque no Portfólio

O portfólio destaca uma série de aplicações desenvolvidas por mim, cobrindo diferentes áreas do ecossistema de desenvolvimento (Frontend, Backend e Full Stack):

| Projeto | Descrição | Tecnologias |
| :--- | :--- | :--- |
| **inventario-bobinas** | Sistema de conciliação de estoque física/digital (SAP) com tríplice entrada: scanner USB, câmera móvel e processamento de Drone via jsQR + Supabase. | JS, React, Supabase, jsQR, CSS |
| **mais-clinica** | Plataforma completa para gestão de clínicas médicas e odontológicas, com controle de agendamentos e prontuários. | JS, React, Bootstrap, Node.js |
| **MoneyBoard** | Aplicação web inteligente para controle e planejamento financeiro pessoal de forma intuitiva. | JS, React, Bootstrap, Node.js |

| **Geek Loop** | Plataforma interativa desenvolvida com foco no público e cultura pop/geek. | JS, React, Bootstrap |
| **Floréssia Pratas** | Vitrine virtual e e-commerce elegante desenvolvido com design moderno para joias de prata. | JS, React, Bootstrap |
| **bom-burguer** | Sistema digital de pedidos e cardápio interativo online para hamburguerias. | JS, React, Bootstrap |
| **sei** | Aplicação web frontend com forte foco em design responsivo e visualização fluida de dados. | JS, React, Bootstrap, Node.js |
